# Jefferson County, WV - Sample Homepage Demo

This is a conceptual sample homepage provided for RFP evaluation purposes. It demonstrates improved usability and modern web standards while respecting the existing Jefferson County brand identity.

## Purpose

This demo showcases:
- Clear civic information hierarchy
- Improved usability and scannability
- Modern, accessible layout
- Respect for existing county branding

**Note:** This is a demonstration homepage only, not a full production site. Final structure and content would be refined post-award.

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **pnpm** package manager

## Getting Started

### Installation

```bash
pnpm install
```

### Development

```bash
pnpm dev
```

Open [http://localhost:3000](http://localhost:3000) to view the homepage.

### Build

```bash
pnpm build
pnpm start
```

## Brand Colors

## Accessibility

The demo follows web accessibility best practices:
- Semantic HTML structure
- Proper heading hierarchy
- High contrast color combinations
- Visible focus states
- ARIA labels for screen readers
- Keyboard navigation support
